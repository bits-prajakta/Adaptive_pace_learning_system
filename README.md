# Adaptive Pace Learning System

Adaptive Pace Learning System is a personalized learning platform built using Python, Machine Learning, and Streamlit. It provides customized learning experiences by adapting content and recommendations according to each user's learning speed, accuracy, and progress.

Users can create an account, search for topics, and take interactive quizzes containing unique questions. Based on quiz performance, response time, and attempts, a machine learning model classifies learners into different pace categories and generates a personalized learning path.

The system continuously tracks user progress using time-based modeling and recommends suitable learning resources such as articles, websites, PDFs, and videos. It also supports collaborative learning by allowing users to join or create group study sessions with learners studying similar topics.

Adaptive quizzes are generated periodically to assess improvement and update learning recommendations. Interactive dashboards help users monitor their progress, accuracy, and topic completion. The platform supports multiple topics simultaneously, creating an intelligent and adaptive learning environment that enhances learning according to each user's pace.
